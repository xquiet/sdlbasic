Toolbar

Here below you'll find a brief description of the Toolbar commands.
New: create a new empty project in a new buffers. See buffers section for more details
Open: Open, a previously saved program, from disk to a new buffer.
Save: program in current buffer to disk.
Close: close current buffer, be aware that unsaved programs will be lost.
Undo: Unmake changes made on text
Redo: Restore changes made on text
Find: Open search window dialog.
Find next: find next occurrence of previously searched text.
Replace: Open find and replace window dialog.
Toggle bookmark: Set a bookmark on current line.
Previous bookmark: Jump to previous bookmark.
Next bookmark: Jump to next bookmark.
Go: Save current buffer program and run it.
Stop: Terminate current program execution (if any).
