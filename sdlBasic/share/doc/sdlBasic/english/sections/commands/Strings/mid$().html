mid$()

Return string with chars 1..n from source. You can omit the dollar character so you can alternatively write it as mid() .
Syntax:
string = mid$( string , start , {end} )
string = mid( string , start , {end} )

Example:
a$="| hello |"
print( mid$( a$,4,4 ) )
print( mid$( a$,4 ) )

See also left$() and right$() commands.
