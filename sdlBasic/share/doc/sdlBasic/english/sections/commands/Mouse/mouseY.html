mouseY

Returns absolute Y mouse coordinate on display. Old Ymouse syntax is still valid but discouraged.

 Syntax:
variable = mouseY

Example:
while not key(k_esc)
    locate(0,0)
    prints( "Current X mouse coordinate is: " & mouseX )
    prints( "Current Y mouse coordinate is: " & mouseY )
    waitVBL
    cls
wend

See also mouseScreenX(), mouseScreenY() and mouseX commands.
