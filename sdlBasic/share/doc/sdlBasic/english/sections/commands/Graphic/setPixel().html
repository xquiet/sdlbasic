setPixel()

Write a dot, with current ink color, on current screen at given coordinates.
Alternatively you can use the dot() or putpixel() commands syntax.

 Syntax:
setPixel( x , y )
putPixel ( x , y )
dot( x , y )

Example:
for n = 1 to 10000
ink( rgb(rnd(255),rnd(255),rnd(255)) )
dot( rnd(640), rnd(480) )
next
waitKey

Example:
3d_stars.sdlbas

See also ink() command.
